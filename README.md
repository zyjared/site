## 部署

确定项目文件夹

```bash
mkdir site
cd site
```

下载部署脚本

```bash
curl -O https://raw.githubusercontent.com/zyjared/site/main/deploy.sh
sudo chmod +x deploy.sh
sudo ./deploy.sh
```
