# Floyd-Warshall 算法

## 介绍

Floyd-Warshall 算法是一种用于计算图中所有节点对之间最短路径的算法，适用于有负权边的图。与其他最短路径算法不同，Floyd-Warshall 算法计算的是图中每对节点之间的最短路径，而不仅仅是单源最短路径。该算法使用动态规划的思想，通过逐步检查每个中间节点来更新路径，最终得到最短路径矩阵。

## 复杂度

|                | 复杂度   | 描述                               |
| -------------- | -------- | ---------------------------------- |
| 最优时间复杂度 | $O(V^3)$ | 由于需要遍历每对节点及所有中间节点 |
| 最坏时间复杂度 | $O(V^3)$ | 在所有节点对间更新最短路径         |
| 平均时间复杂度 | $O(V^3)$ | 时间复杂度对节点数立方关系         |
| 空间复杂度     | $O(V^2)$ | 存储所有节点对的最短路径           |

## 特点

### 优点

- 可以计算任意两个节点之间的最短路径，适用于多源最短路径问题。
- 能够处理有负权边的图。
- 算法简单且直观，通过动态规划更新最短路径。

### 缺点

- 时间复杂度为 $O(V^3)$，对于大规模图来说，效率较低。
- 不适合稀疏图，计算复杂度较高。

## 应用场景

Floyd-Warshall 算法适用于以下场景：

- 需要计算图中任意两点间最短路径的场景。
- 网络优化问题，尤其是在多源最短路径的情况下。
- 在某些情况下，用于检测图中是否存在负权回路。
- 其他涉及图中节点对间最短路径计算的应用。
