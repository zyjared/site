# 深度优先搜索

## 介绍

深度优先搜索（DFS）是一种用于遍历或搜索树或图的算法。它从根节点开始，沿着树的深度遍历尽可能深的节点，直到没有未访问的节点为止，然后回溯并继续遍历其他节点。DFS 是一种递归算法，也可以通过栈来实现。

## 特点

### 优点

- 算法简单，容易实现。
- 可以用递归方式实现，代码简洁。
- 在图较为稀疏的情况下效率较高。

### 缺点

- 如果图很深，可能会导致栈溢出（对于递归实现）。
- 可能会遍历到某些不必要的节点，效率相对较低，尤其在图很大的时候。
- 不适合求解最短路径问题，因为它不会优先选择较短的路径。

## 应用场景

深度优先搜索常用于以下场景：

- 求解图的连通性。
- 拓扑排序。
- 寻找图中的所有路径。
- 解决迷宫问题、棋盘问题等。
- 二叉树的遍历（前序、中序、后序遍历）。
