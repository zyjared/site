# 插值查找

## 介绍

插值查找是一种改进版的二分查找算法，适用于值较为均匀分布的有序数组。与二分查找不同，插值查找通过估算目标元素可能的位置来进行查找，因此在某些情况下，它的性能比二分查找更好。

## 复杂度

|                | 复杂度           | 描述                             |
| -------------- | ---------------- | -------------------------------- |
| 最优时间复杂度 | $O(\log \log n)$ | 元素分布均匀时表现最优           |
| 最坏时间复杂度 | $O(n)$           | 元素分布不均匀时，退化为线性查找 |
| 平均时间复杂度 | $O(\log \log n)$ | 元素分布均匀时表现良好           |
| 空间复杂度     | $O(1)$           | 只需要常数级空间                 |

## 特点

### 优点

- 在元素分布均匀的情况下，比二分查找更高效，时间复杂度更低。
- 与二分查找相比，插值查找对数组的访问更加灵活，不仅仅依赖于中间元素。

### 缺点

- 仅适用于值分布均匀的有序数组。
- 如果数据分布不均匀，插值查找的效率可能较差，甚至退化为线性查找。

## 应用场景

插值查找适用于查找范围较大且数据分布均匀的有序数组，常用于类似数值型数据的查找，比如快速查找股票价格、温度等数据。
