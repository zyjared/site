# 冒泡排序

## 介绍

冒泡排序（Bubble Sort）是一种简单的排序算法。重复走访要排序的元素，一次比较两个元素，如果顺序错误就交换，最大（或最小）的元素总会被移动到序列的末尾。

## 复杂度

|                | 复杂度   | 描述                             |
| -------------- | -------- | -------------------------------- |
| 最优时间复杂度 | $O(n)$   | 已经排序好时（可以提前结束）     |
| 最坏时间复杂度 | $O(n^2)$ | 正好是逆序时，需要最大次数的交换 |
| 平均时间复杂度 | $O(n^2)$ | 每个元素大致需要 $n$ 次比较      |
| 空间复杂度     | $O(1)$   | 原地排序，不需要额外空间         |

## 特点

### 优点

- 简单实现，原地排序，不需要额外空间。

### 缺点

- 效率较低，适用于小规模数据集。

## 应用场景

冒泡排序适用于对小规模数据进行排序，但不适用于大规模数据集，因其时间复杂度较高。
