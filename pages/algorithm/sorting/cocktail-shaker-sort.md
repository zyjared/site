# 鸡尾酒排序

## 介绍

鸡尾酒排序（Cocktail shaker sort）是一种双向冒泡排序算法，改进自冒泡排序。与冒泡排序不同，鸡尾酒排序在每一轮遍历时不仅从左到右遍历，还从右到左遍历。这样可以减少某些情况下的比较次数，提升排序效率。它通过两次冒泡操作逐步把最大和最小的元素交换到合适的位置。

鸡尾酒排序是稳定的排序算法。

## 复杂度

|                | 复杂度   | 描述                                                |
| -------------- | -------- | --------------------------------------------------- |
| 最优时间复杂度 | $O(n)$   | 如果数组已经有序，只有一次遍历即可完成排序。        |
| 最坏时间复杂度 | $O(n^2)$ | 当数组完全逆序时，需要进行 $O(n^2)$ 次操作。        |
| 平均时间复杂度 | $O(n^2)$ | 时间复杂度接近冒泡排序，在大多数情况下为 $O(n^2)$。 |
| 空间复杂度     | $O(1)$   | 采用原地排序，不需要额外的空间。                    |

## 特点

### 优点

- 稳定排序，保持相等元素的相对顺序。
- 相较于普通的冒泡排序，鸡尾酒排序在某些情况下可能减少比较次数，提升排序效率。

### 缺点

- 时间复杂度在大部分情况下仍然为 $O(n^2)$，对于大规模数据集，性能较差。
- 实现较为复杂，适用场景有限。

## 应用场景

鸡尾酒排序适用于小规模数据集的排序，它是冒泡排序的一种改进，对于已经接近有序的数据集有较好的性能。在数据量较大时，通常会选择更高效的排序算法（如快速排序、归并排序等）。
