# 计数排序

## 介绍

计数排序（Counting Sort）是一种非比较型排序算法。它通过统计数组中每个元素出现的次数，进而直接确定元素在排序结果中的位置。适用于元素范围较小的情况，但对于范围较大的数据集不适用。

| 最优时间复杂度 | $O(n+k)$ | 计数数组的构建           |
| 最坏时间复杂度 | $O(n+k)$ | 计数数组的构建           |
| 平均时间复杂度 | $O(n+k)$ | 计数数组的构建           |
| 空间复杂度     | $O(n+k)$ | 额外的计数和输出数组空间 |

## 特点

### 优点

- 稳定排序，不会改变相同元素的相对顺序。
- 时间复杂度为 $O(n+k)$，在合适的情况下可以非常高效。

### 缺点

- 当元素范围较大时，效率较低，内存消耗较高。
- 只能处理整数或有限范围的离散值。

## 应用场景

计数排序适用于处理整数或有限范围的离散数据，尤其在数据范围较小的情况下表现出色。常用于排序学生成绩、电话号码等。
