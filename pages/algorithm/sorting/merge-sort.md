# 归并排序

## 介绍

归并排序（Merge Sort）是一种基于分治法的排序算法。它将数组分为两半，分别排序这两部分，然后将它们合并成一个有序的数组。归并排序的关键是“归并”操作，将两个已排序的子数组合并成一个有序的数组。

归并排序是一种稳定的排序算法，且在最坏情况下的时间复杂度为 $O(n \log n)$。

## 特点

### 优点

- 稳定排序，排序过程不会改变相等元素的相对顺序。
- 在最坏情况下的时间复杂度是 $O(n \log n)$，适用于大规模数据排序。
- 分治法使得算法易于并行化，可以更高效地处理大数据集。

### 缺点

- 空间复杂度为 $O(n)$，需要额外的空间存储临时数据，因此在内存受限的情况下不适用。
- 不适合用于小规模数据排序，效率可能不如其他简单的排序算法。

## 应用场景

归并排序适用于大规模数据排序，尤其是在数据集较大且内存充足的情况下。它常用于外部排序，如在磁盘上的大数据集排序，也适合并行计算。
