# 快速排序

## 介绍

快速排序是一种分治法排序算法，通过选择一个基准元素（pivot），将数据集分为两个子集，左侧子集的元素都小于基准，右侧子集的元素都大于基准。然后递归地对左右子集进行排序，最终实现整个数据集的排序。

快速排序通常比其他算法（如冒泡排序和选择排序）更高效，尤其是在大规模数据集时。

## 复杂度

|                | 复杂度        | 描述                                 |
| -------------- | ------------- | ------------------------------------ |
| 最优时间复杂度 | $O(n \log n)$ | 基准元素能均匀地将数据分成两半       |
| 最坏时间复杂度 | $O(n^2)$      | 基准选择不当，导致每次划分都不平衡   |
| 平均时间复杂度 | $O(n \log n)$ | 基准元素平均能够将数据分成均匀的两半 |

## 特点

### 优点

- 分治法，可以更高效地处理大数据集。
- 通常比其他排序算法（如冒泡排序和选择排序）更高效，尤其是在大规模数据集时。

### 缺点

- 对于有序数据，快速排序的时间复杂度为 $O(n^2)$，性能较差。

## 应用场景

快速排序适用于大规模数据排序，尤其是在数据分布较为均匀的情况下，它的效率远超其他 $O(n^2)$ 算法。由于其分治性质，快速排序也很适合用于并行计算。
